# Airline-Flight-Delay-Analysis 
https://www.kaggle.com/datasets/daryaheyko/airline-on-time-statistics-and-delay-causes-bts  
# number of rows >= 300,000 
 # Variable  Name Type  	Description
year	Integer	The year the flight took place (e.g., 2024).
month	Integer	The month of the year (1 = Jan ,12 = Dec).
carrier	String	The unique 2-letter airline code (e.g., DL for Delta, UA for United).
carrier_name	String	The full name of the airline reporting the data.
airport	String	The 3-letter IATA code for the arrival airport (e.g., JFK, ORD).
airport_name	String	The full name/location of the airport.
arr_flights	Integer	Total Arrivals: The total number of flights scheduled to land.
arr_del15	Integer	Delayed Count: Number of flights arriving 15+ minutes late.
arr_cancelled	Integer	Cancelled: Number of flights that never flew.
arr_diverted	Integer	Diverted: Flights that landed at an airport they weren't supposed to.
